Subject & BehaviorSubject

Both Are are Multicasting Observables Means They allow multiple subscribers to listen to the same Observable stream. However, they differ in how they handle emissions, especially initial values and replay behavior.

Subject vs BehaviorSubject (RxJS)

Feature Subject BehaviorSubject
Requires initial value No ❌ Yes ✅
Emits last value to new subscribers No ❌ Yes ✅
Use case Events (e.g. clicks, notifications) State/data stream (e.g. auth status, current user)
Replay old values Never ❌ Always (latest only) ✅
